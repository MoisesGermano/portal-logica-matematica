# 🧠 Portal de Lógica Matemática

🌐 **Acesse a aplicação online:** [app-logica-matematica.streamlit.app](https://app-logica-matematica-hcpqrteqew9y2eunwkm94s.streamlit.app/)

## 📖 Sobre o Projeto
Este portal é um ambiente de estudo interativo focado em Lógica Matemática e Computacional. Desenvolvido para consolidar os fundamentos lógicos que são a base essencial para o dia a dia de um **Cientista de Dados**, o projeto transforma anotações e atividades acadêmicas (como as da Fatec) em uma aplicação web dinâmica. 

O domínio de tabelas-verdade, álgebra proposicional e formas normais é indispensável para a otimização de algoritmos complexos, estruturação de queries SQL avançadas e tomada de decisão orientada a dados.

## 🚀 Funcionalidades
A plataforma é dividida em módulos de fácil navegação, contendo resumos teóricos e resoluções práticas em vídeo:
* **Tabela-Verdade:** Fundamentos de conectivos lógicos (Conjunção, Disjunção, Condicional).
* **FND e FNC:** Padronização de fórmulas através da Forma Normal Disjuntiva e Conjuntiva.
* **Negação de Proposições:** Regras práticas para inverter valores lógicos de sentenças compostas.
* **Leis de Morgan:** Técnicas de manipulação algébrica para simplificação de argumentos.
* **Exercícios:** Resoluções detalhadas de atividades teóricas e enigmas lógicos.
* **Mapa Mental:** Visualização estruturada e interativa do fluxo de estudos da disciplina.

## 🛠️ Tecnologias Utilizadas
* **Python:** Linguagem base da aplicação.
* **Streamlit:** Framework para a construção da interface web e roteamento das páginas.
* **Graphviz:** Geração nativa e renderização do mapa mental interativo.
* **HTML5 / CSS3:** Injeção de estilos customizados para aprimoramento de UI/UX (botões interativos, sombras, animações de pulso e responsividade).
